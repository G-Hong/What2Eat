# What2Eat

## 목표와 범위

- 다이어트 시작일, 시작체중/체지방/근육량을 기반으로 현재 다이어트 단계 계산 및 띄우기
- 현재 다이어트 단계를 기반으로 허용식품에 대한 질문 처리
